# Visualizing Housing Market Trends: An Analysis of Sale Prices and Features using Tableau
# Project Description

This project analyzes housing market data to identify the factors influencing property sale prices using Tableau. It transforms raw housing datasets into interactive dashboards and visualizations to reveal market trends and patterns. The insights generated help buyers, investors, and real estate professionals make informed, data-driven decisions.
