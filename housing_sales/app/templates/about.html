📖 Project Overview

About This Project

A comprehensive housing market analytics platform built to uncover the key factors influencing residential property prices and sales trends in King County, Washington.

Project Description

This project focuses on uncovering the key factors that influence housing prices and sales trends by analyzing a comprehensive dataset of 21,588 residential properties in King County, WA. Using interactive data visualization as the primary tool, the analysis examines how variables such as years since renovation, house age, number of bathrooms, bedrooms, and floors relate to overall market behavior.

A central aim is to assess the impact of renovations on property value and age distribution. Through interactive dashboards and visual storytelling, the project delivers actionable insights that support strategic decision-making, optimize pricing models, and enhance market competitiveness.

🗂 Methodology

Project Phases

Phase 1

Data Collection

King County House Sales dataset (kc_house_data.csv) — 21,613 records, 22 columns, May 2014–May 2015. Sourced from IBM CDN / Kaggle.

Phase 2

Data Preparation

Cleaned 25 NaN rows, removed outliers (price < $50K or > $5M, bedrooms > 10). Created 7 calculated fields using Python/Pandas.

Phase 3

Visualizations (7 Charts)

Bar, Histogram, Scatter, and Line charts covering renovation impact, age distribution, room effects, and price trends. Built with Plotly.

Phase 4

Dashboard

Interactive dashboard with 5 live filters (renovation, bedrooms, floors, price range, year built) updating all 7 charts simultaneously.

Phase 5

Storyboard

5-chapter visual narrative — Market Landscape → Renovation Impact → Age Story → Room Impact → Recommendations. Scroll-animated.

Phase 6

Performance Testing

All 5 data filters validated. All 7 calculation fields verified. Pre-aggregated JSON API for fast load times. Responsive at 375px → 1440px.

Phase 7

Web Integration

Flask app with 5 routes + JSON API. Tableau Embedding API v3 integration page for live Tableau Public embed support.

🧮 Calculated Fields

7 Calculated Fields

# Field Name Description Tableau Formula
1 house_age Years since the house was built (as of 2015) 2015 - [yr_built]
2 years_since_renovation Years since last renovation; uses house_age if never renovated IF [yr_renovated] > 0 THEN 2015 - [yr_renovated] ELSE [house_age] END
3 renovation_status Classifies each property as Renovated or Never Renovated IF [yr_renovated] > 0 THEN "Renovated" ELSE "Never Renovated" END
4 price_per_sqft Price efficiency metric — price per square foot of living space [price] / [sqft_living]
5 age_category Bucketed age groups for histogram analysis IF [house_age] < 10 THEN "New" ELSEIF < 30 THEN "Mid-age" ... END
6 bathroom_category Bathroom tier grouping (1 / 2 / 3+) IF [bathrooms] ≤ 1 THEN "1 Bath" ELSEIF ≤ 2 THEN "2 Bath" ELSE "3+" END
7 bedroom_category Bedroom tier grouping (1–2 / 3 / 4 / 5+) IF [bedrooms] ≤ 2 THEN "1–2 BR" ELSEIF = 3 THEN "3 BR" ... END
