📋 How To

Publishing & Embedding Guide

Follow these steps to publish your Tableau workbook and embed it here.

1

Connect the Data

Open Tableau Desktop → Connect to Data → Text File → Select kc_house_data_prepared.csv from your /data/ folder.

2

Create Calculated Fields

In Tableau, right-click the data panel → Create Calculated Field. Add the 7 fields documented in TABLEAU_GUIDE.md.

Analysis → Create Calculated Field
3

Build Visualizations

Create 7 worksheets (1 per visualization). Drag dimensions to Rows/Columns and measures to the Marks shelf.

4

Assemble Dashboard

Go to Dashboard → New Dashboard. Drag worksheets into a layout container. Add filter actions for interactivity.

5

Create Story

Go to Story → New Story. Add 5 story points, one per chapter. Annotate each slide with captions.

6

Publish & Embed

Server → Tableau Public → Save to Tableau Public. Copy the URL and paste it in the embed box above.

File → Tableau Public → Save

Tableau Embedding API v3 — Code Snippet

This is the exact HTML used to embed your Tableau view using the official <tableau-viz> web component:

<!-- 1. Load the Embedding API -->
<script type="module"
  src="https://public.tableau.com/javascripts/api/tableau.embedding.3.latest.min.js"></script>

<!-- 2. Place the web component -->
<tableau-viz
  id="tableauViz"
  src="https://public.tableau.com/views/YOUR_WORKBOOK/YOUR_DASHBOARD"
  toolbar="bottom"
  hide-tabs
></tableau-viz>
